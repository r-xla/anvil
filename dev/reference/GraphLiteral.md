# Graph Literal

Literal in an
[`AnvlGraph`](https://r-xla.github.io/anvl/dev/reference/AnvlGraph.md).
This is a mutable class.

## Usage

``` r
GraphLiteral(aval)
```

## Arguments

- aval:

  ([`LiteralArray`](https://r-xla.github.io/anvl/dev/reference/LiteralArray.md))  
  The value of the literal.

## Value

(`GraphLiteral`)
