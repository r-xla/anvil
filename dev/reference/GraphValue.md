# Graph Value

Value in an
[`AnvlGraph`](https://r-xla.github.io/anvl/dev/reference/AnvlGraph.md).
This is a mutable class.

## Usage

``` r
GraphValue(aval)
```

## Arguments

- aval:

  ([`AbstractArray`](https://r-xla.github.io/anvl/dev/reference/AbstractArray.md))  
  The abstract value of the variable.
