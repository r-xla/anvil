# Logical Shift Right

Element-wise bitwise logical right shift.

## Usage

``` r
nv_shift_right_logical(lhs, rhs)
```

## Arguments

- lhs, rhs:

  ([`tensorish`](https://r-xla.github.io/anvil/reference/tensorish.md))  
  Left and right operand.
