# Graph Node

Virtual base class for nodes in an
[`AnvlGraph`](https://r-xla.github.io/anvl/reference/AnvlGraph.md). Is
either a
[`GraphValue`](https://r-xla.github.io/anvl/reference/GraphValue.md) or
a
[`GraphLiteral`](https://r-xla.github.io/anvl/reference/GraphLiteral.md).
Cannot be instantiated directly - use
[`GraphValue()`](https://r-xla.github.io/anvl/reference/GraphValue.md)
or
[`GraphLiteral()`](https://r-xla.github.io/anvl/reference/GraphLiteral.md)
instead.
