# Primitive Power

Raises lhs to the power of rhs element-wise.

## Usage

``` r
nvl_pow(lhs, rhs)
```

## Arguments

- lhs, rhs:

  ([`tensorish`](https://r-xla.github.io/anvil/reference/tensorish.md))  
  Left and right operand.

## Value

[`tensorish`](https://r-xla.github.io/anvil/reference/tensorish.md)
