# Arctangent 2

Element-wise two-argument arctangent.

## Usage

``` r
nv_atan2(lhs, rhs)
```

## Arguments

- lhs, rhs:

  ([`tensorish`](https://r-xla.github.io/anvil/reference/tensorish.md))  
  Left and right operand.

[`tensorish`](https://r-xla.github.io/anvil/reference/tensorish.md)
