# Primitive Shift Left

Element-wise left bit shift.

## Usage

``` r
nvl_shift_left(lhs, rhs)
```

## Arguments

- lhs, rhs:

  ([`tensorish`](https://r-xla.github.io/anvil/reference/tensorish.md))  
  Left and right operand.

## Value

[`tensorish`](https://r-xla.github.io/anvil/reference/tensorish.md)
