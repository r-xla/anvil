# Primitive Not Equal

Element-wise inequality comparison.

## Usage

``` r
nvl_ne(lhs, rhs)
```

## Arguments

- lhs, rhs:

  ([`tensorish`](https://r-xla.github.io/anvil/reference/tensorish.md))  
  Left and right operand.

## Value

[`tensorish`](https://r-xla.github.io/anvil/reference/tensorish.md)
(boolean)
