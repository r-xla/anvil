# AnvilPrimitive

Primitive interpretation rule. Note that `[[` and `[[<-` access the
interpretation rules. To access other fields, use `$` and `$<-`.

## Usage

``` r
AnvilPrimitive(name)
```

## Arguments

- name:

  ([`character()`](https://rdrr.io/r/base/character.html))  
  The name of the primitive.

## Value

(`AnvilPrimitive`)
