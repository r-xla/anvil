# Random Uniform Numbers

generate random uniform numbers

## Usage

``` r
nv_runif(initial_state, dtype = "f64", shape_out, lower = 0, upper = 1)
```

## Arguments

- initial_state:

  state seed

- dtype:

  output dtype either "f32" or "f64"

- shape_out:

  output shape

- lower:

  lower bound

- upper:

  upper bound
