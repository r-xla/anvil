# Primitive Division

Divides two tensors element-wise.

## Usage

``` r
nvl_div(lhs, rhs)
```

## Arguments

- lhs, rhs:

  ([`tensorish`](https://r-xla.github.io/anvil/reference/tensorish.md))  
  Left and right operand.

## Value

[`tensorish`](https://r-xla.github.io/anvil/reference/tensorish.md)
