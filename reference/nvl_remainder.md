# Primitive Remainder

Element-wise remainder of division.

## Usage

``` r
nvl_remainder(lhs, rhs)
```

## Arguments

- lhs, rhs:

  ([`tensorish`](https://r-xla.github.io/anvil/reference/tensorish.md))  
  Left and right operand.

## Value

[`tensorish`](https://r-xla.github.io/anvil/reference/tensorish.md)
