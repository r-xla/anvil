# Graph Box

Box that represents a node in a [`GraphDescriptor`](GraphDescriptor.md).

## Usage

``` r
GraphBox(gnode, desc)
```

## Arguments

- gnode:

  ([`GraphNode`](GraphNode.md))  
  The node.

- desc:

  ([`GraphDescriptor`](GraphDescriptor.md))  
  The descriptor of the graph.

## Value

(`GraphBox`)
