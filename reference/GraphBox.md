# Graph Box

Box that represents a node in a
[`GraphDescriptor`](https://r-xla.github.io/anvil/reference/GraphDescriptor.md).

## Usage

``` r
GraphBox(gnode, desc)
```

## Arguments

- gnode:

  ([`GraphNode`](https://r-xla.github.io/anvil/reference/GraphNode.md))  
  The node.

- desc:

  ([`GraphDescriptor`](https://r-xla.github.io/anvil/reference/GraphDescriptor.md))  
  The descriptor of the graph.

## Value

(`GraphBox`)
