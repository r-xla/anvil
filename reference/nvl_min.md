# Primitive Minimum

Element-wise minimum of two tensors.

## Usage

``` r
nvl_min(lhs, rhs)
```

## Arguments

- lhs, rhs:

  ([`tensorish`](https://r-xla.github.io/anvil/reference/tensorish.md))  
  Left and right operand.
