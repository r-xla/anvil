# Reverse

Reverses the order of elements along specified dimensions.

## Usage

``` r
nv_reverse(operand, dimensions)
```

## Arguments

- operand:

  ([`tensorish`](tensorish.md))  
  Operand.

- dimensions:

  ([`integer()`](https://rdrr.io/r/base/integer.html))  
  Dimensions to reverse (1-indexed).

## Value

[`tensorish`](tensorish.md)
