# Primitive Arithmetic Shift Right

Element-wise arithmetic right bit shift.

## Usage

``` r
nvl_shift_right_arithmetic(lhs, rhs)
```

## Arguments

- lhs, rhs:

  ([`tensorish`](https://r-xla.github.io/anvil/reference/tensorish.md))  
  Left and right operand.
