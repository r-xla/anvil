# If

Functional if statement.

## Usage

``` r
nv_if(pred, true, false)
```

## Arguments

- pred:

  ([`nv_tensor`](nv_tensor.md))  
  Flag.

- true:

  (NSE)  
  Expression to evaluate if the condition is true.

- false:

  (NSE)  
  Expression to evaluate if the condition is false.

## Value

[`nv_tensor`](nv_tensor.md)
