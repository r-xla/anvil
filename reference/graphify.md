# Graphify an R function

Graphify an R function.

## Usage

``` r
graphify(f, args, desc = NULL)
```

## Arguments

- f:

  (`function`)  
  The function to graphify.

- args:

  (`list`)  
  The arguments to the function.

- desc:

  (`NULL` \| `GraphDescriptor`)  
  The descriptor to use for the graph.
