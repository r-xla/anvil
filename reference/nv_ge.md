# Greater Than or Equal

Element-wise greater than or equal comparison.

## Usage

``` r
nv_ge(lhs, rhs)
```

## Arguments

- lhs, rhs:

  ([`tensorish`](https://r-xla.github.io/anvil/reference/tensorish.md))  
  Left and right operand.
