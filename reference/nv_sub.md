# Subtraction

Element-wise subtraction of two tensors.

## Usage

``` r
nv_sub(lhs, rhs)
```

## Arguments

- lhs, rhs:

  ([`tensorish`](https://r-xla.github.io/anvil/reference/tensorish.md))  
  Left and right operand.
