# Broadcast

Broadcast a tensor to a given shape using NumPy broadcasting rules.

## Usage

``` r
nv_broadcast_to(operand, shape)
```

## Arguments

- operand:

  ([`tensorish`](https://r-xla.github.io/anvil/reference/tensorish.md))  
  Operand.

- shape:

  ([`integer()`](https://rdrr.io/r/base/integer.html))  
  Output shape.

## Value

([`tensorish`](https://r-xla.github.io/anvil/reference/tensorish.md))
