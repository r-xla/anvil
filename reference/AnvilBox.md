# AnvilBox

Virtual base class for GraphBox and DebugBox. This class is used to
represent values during graph construction and debugging. Cannot be
instantiated directly - use
[`GraphBox()`](https://r-xla.github.io/anvil/reference/GraphBox.md) or
[`DebugBox()`](https://r-xla.github.io/anvil/reference/DebugBox.md)
instead.
