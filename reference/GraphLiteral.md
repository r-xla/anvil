# Graph Literal

Literal in a [`Graph`](Graph.md).

## Usage

``` r
GraphLiteral(aval = NULL, dtype = stablehlo::BooleanType())
```

## Arguments

- aval:

  (`any`)  
  The value of the literal.

- dtype:

  (`character(1)` \|
  [`stablehlo::TensorDataType`](https://r-xla.github.io/stablehlo/reference/TensorDataType.html))  
  Data type.
