# Graph Literal

Literal in an
[`AnvilGraph`](https://r-xla.github.io/anvil/reference/AnvilGraph.md).
This is a mutable class.

## Usage

``` r
GraphLiteral(aval)
```

## Arguments

- aval:

  ([`LiteralTensor`](https://r-xla.github.io/anvil/reference/LiteralTensor.md))  
  The value of the literal.

## Value

(`GraphLiteral`)
