# Primitive Print

Prints a tensor during execution. Returns the input unchanged. Note:
Currently only works on CPU backend.

## Usage

``` r
nvl_print(operand)
```

## Arguments

- operand:

  ([`tensorish`](https://r-xla.github.io/anvil/reference/tensorish.md))  
  Operand.

## Value

[`tensorish`](https://r-xla.github.io/anvil/reference/tensorish.md)
