# Binary Operations

Binary operations on tensors.

## Usage

``` r
nv_add(lhs, rhs)

nv_mul(lhs, rhs)

nv_sub(lhs, rhs)

nv_div(lhs, rhs)

nv_pow(lhs, rhs)

nv_eq(lhs, rhs)

nv_ne(lhs, rhs)

nv_gt(lhs, rhs)

nv_ge(lhs, rhs)

nv_lt(lhs, rhs)

nv_le(lhs, rhs)

nv_max(lhs, rhs)

nv_min(lhs, rhs)

nv_remainder(lhs, rhs)

nv_and(lhs, rhs)

nv_or(lhs, rhs)

nv_xor(lhs, rhs)

nv_shift_left(lhs, rhs)

nv_shift_right_logical(lhs, rhs)

nv_shift_right_arithmetic(lhs, rhs)

nv_atan2(lhs, rhs)
```

## Arguments

- lhs:

  ([`nv_tensor`](nv_tensor.md))

- rhs:

  ([`nv_tensor`](nv_tensor.md))

## Value

[`nv_tensor`](nv_tensor.md)
